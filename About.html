Amplify uses GPT-3 AI technology to amplify your text. Use it to finish incomplete sentences, add to lists, flesh out paragraphs, answer questions, and more.

Select a segment of text, click "Amplify your text," and watch it work. The newly added text will be selected in the document.

You can also use select text the "Generate an image" function to drop in a DALL-E AI-generated image based on a prompt you enter. Once a image is inserted, the original text stays selected in case you want to do it again.

You'll need to add an OpenAI API key to Settings to begin.

Amplify is open source software released under the GPL 2.0 license. View the source code on GitHub. There's also a Privacy Policy and Terms of Use, which says that I'm not collecting your data and it's not my fault if something breaks.
